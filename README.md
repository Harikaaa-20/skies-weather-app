# ☁️ Skies — Weather App

A clean, minimal weather app built with HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Shows temperature, humidity, wind speed, pressure, visibility
- Weather emoji based on conditions
- API key saved locally so you only enter it once

## Setup
1. Get a free API key from [openweathermap.org](https://openweathermap.org/api)
2. Open `index.html` in your browser via a local server
3. Paste your API key and search any city

## Run locally
```bash
python3 -m http.server 8080
```
Then open `http://localhost:8080`

## Tech
- Vanilla HTML / CSS / JS
- OpenWeatherMap API
